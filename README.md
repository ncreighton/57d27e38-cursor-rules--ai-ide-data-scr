# Cursor Rules & AI IDE Data Scraper and Exporter Chrome Extension

Extract structured data from Cursor Rules & AI IDE websites with one click, export to CSV/JSON/Sheets. Saves hours of manual data collection.

## Installation

### Chrome
1. Download `chrome_extension.zip`
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download `firefox_extension.zip`
2. Go to `about:debugging`
3. Click "Load Temporary Add-on"
